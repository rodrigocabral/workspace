Exemplo de JPA 
==============

Esse é um exemplo básico da utilização da biblioteca JPA utilizando uma base .mdb (MS Access) e o driver JBDC-ODBC. O código foi escrito utilizando a IDE Eclipse.

Observações
-----------

* Para mudar o banco de dados assegure-se de instalar o driver JDBC correto
* Altere as configurações do banco de dados no arquivo persistence.xml

Testando
--------

* Abra o Eclipse
* File -> Import -> General -> Existing Projects Into Workspace
* Altere as configurações do banco de dados no arquivo persistence.xml
* Execute

Contribuindo
------------

1. Faça um fork do projeto no github.
2. Crie um branch (`git checkout -b meu_branch`)
3. Commite suas alterações (`git commit -am "seus comentários"`)
4. Envie (push) para o repositório central (`git push origin meu_branch`)
5. Crie um [Issue] com um link para o seu branch
6. Espere até a aprovação pelo dono do repositório
